#  Project Overview

This repository contains my **Digital Literacy Project** completed as part of the **CSE0001 course at VIT Bhopal University**.

As a **Student Digital Ambassador**, the objective of this project was to develop awareness about digital tools, professional communication, online safety, and responsible internet usage.

The project is divided into **five tasks**, each focusing on a specific aspect of digital literacy. All required files, screenshots, and the final report are included in this repository.
